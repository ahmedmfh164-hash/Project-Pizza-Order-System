# Project-Pizza-Order-System

🍕 Pizza Order System

A simple Windows Forms application built using C# that allows users to create and customize their pizza order.

🚀 Features

- Choose pizza size (Small, Medium, Large)
- Select multiple toppings
- Choose crust type (Thin / Thick)
- Select dining option (Eat In / Take Out)
- Automatically calculate price
- Display full order summary

🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)

"App Screenshot" (screenshot.png)

💡 How It Works

1. Select your pizza size
2. Choose toppings
3. Select crust type
4. Choose where to eat
5. Adjust number of pizzas
6. View total price instantly

▶️ Run the Project

1. Open the solution file "Project Pizza.sln"
2. Run the project using Visual Studio
